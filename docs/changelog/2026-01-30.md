- Tools task creation sets `isByov`, `blockedActions`, `fleetRoutingDecision`
- GET `/api/tools-queue/:id` returns dynamic `currentBlockingStatus`
- Non-BYOV tasks wait for Fleet completion; BYOV tasks bypass wait

**Snowflake Sync (`server/snowflake-sync-service.ts`)**
- BYOV detection integrated into automated Tools task creation
- Sets proper initial state based on truck number
