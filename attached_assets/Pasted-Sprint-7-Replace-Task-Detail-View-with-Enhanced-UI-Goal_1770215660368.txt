Sprint 7: Replace Task Detail View with Enhanced UI
Goal: Create the new 3-column task detail view that replaces the current view when queue='tools'

What You're Building
When a user clicks on a Tools queue task (like "Day 0: Recover Equipment & Tools - ABALOS, JUSTINE W"), they currently see a basic task detail. Replace this with the new enhanced UI.

Integration Point
Find where the task detail view is rendered. Look for:

client/src/pages/ or client/src/components/
Component that renders when clicking "View" on a queue item
Condition: Only replace when queue === 'tools'
